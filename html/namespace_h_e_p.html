Detailed Description

The HEP namespace contains all currently accepted particles within the standard model and beyond. The masses and data are taken from the Particle Data Group listings [1].

Warning
It is recommend you do not use the namespace within the script as there is a high probability of accidental overwriting. Instead use the particles as HEP::Particles::particle
