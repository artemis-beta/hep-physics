\doxysubsection{Detailed Description}
The \mbox{\hyperlink{namespace_p_h_y_s}{P\+H\+YS}} namespace contains all currently accepted particles within the standard model and beyond. The masses and data are taken from the \mbox{\hyperlink{class_p_h_y_s_1_1_particle}{Particle}} \mbox{\hyperlink{namespace_p_h_y_s_1_1_data}{Data}} Group listings \cite{PDGlist}. \begin{DoxyWarning}{Warning}
It is recommend you do not use the namespace within the script as there is a high probability of accidental overwriting. Instead use the particles as P\+H\+Y\+S\+::\+Particles\+::particle
\end{DoxyWarning}
