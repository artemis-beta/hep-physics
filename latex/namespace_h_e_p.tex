\doxysubsection{Detailed Description}
The \mbox{\hyperlink{namespace_h_e_p}{H\+EP}} namespace contains all currently accepted particles within the standard model and beyond. The masses and data are taken from the \mbox{\hyperlink{class_h_e_p_1_1_particle}{Particle}} Data Group listings \cite{PDGlist}. \begin{DoxyWarning}{Warning}
It is recommend you do not use the namespace within the script as there is a high probability of accidental overwriting. Instead use the particles as H\+E\+P\+::\+Particles\+::particle
\end{DoxyWarning}
